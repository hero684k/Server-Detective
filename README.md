## ℹ️ Что это?

**Server Detective** — бесплатный онлайн-инструмент для проверки серверов Minecraft. Узнай статус, онлайн, версию, плагины, MOTD и даже местоположение сервера в один клик.

Никакой рекламы, регистраций и скрытых платежей.

## ✨ Возможности

| Функция | Описание |
|---------|----------|

## 📖 Как пользоваться?

1. Открой сайт по ссылке выше
2. Введи IP сервера (например, `hypixel.net`)
3. Нажми «Проверить»
4. Смотри результат: статус, игроки, версию, MOTD, плагины

Для графика онлайна — проверяй сервер несколько раз в течение дня, затем нажми «📊 График онлайна».

## ❓ Частые вопросы

<details>
<summary><strong>Это бесплатно?</strong></summary>
Да, полностью бесплатно и без рекламы.
</details>

<details>
<summary><strong>Откуда берутся данные?</strong></summary>
Используется публичное API <a href="https://mcsrvstat.us">mcsrvstat.us</a> для статуса серверов и <a href="http://ip-api.com">ip-api.com</a> для геолокации.
</details>

<details>
<summary><strong>Почему график онлайна не сразу строится?</strong></summary>
График строится из сохранённых точек. Проверь сервер несколько раз в течение дня — и график появится.
</details>

<details>
<summary><strong>Мои данные сохраняются?</strong></summary>
Да, история и избранное хранятся в твоём браузере (localStorage). Никакие данные не отправляются на сервер.
</details>

<details>
<summary><strong>Можно ли предложить новый функционал?</strong></summary>
Конечно! Создай Issue или Pull Request на GitHub.
</details>

- Ничего не отправляется на сервер
